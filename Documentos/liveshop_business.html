Unit Economics — Los Números que Importan
Estas métricas determinan si el negocio es sostenible o no
📊 CAC — Costo de Adquisición
CAC = Gasto Marketing / Nuevos Clientes
Bs. 1,500 / 8 clientes = Bs. 187
Bs. 180–200
Meta: reducir a Bs. 120 con referidos orgánicos (mes 6+)
Canales de adquisición:
TikTok Ads (Bs. 800): ~4 clientes → CAC Bs. 200
Referidos (Bs. 0): ~3 clientes → CAC Bs. 0
Orgánico/SEO (Bs. 700): ~1 cliente → CAC Bs. 700
💎 LTV — Valor de Vida del Cliente
LTV = ARPU × (1 / Churn Rate)
Bs. 220 × (1 / 0.06) = Bs. 3,667
Bs. 2,640–3,667
Estimado conservador asumiendo mix de planes y churn del 6%
Cálculo detallado:
Mix: 40% Starter (Bs. 99) + 50% Pro (Bs. 220) + 10% Business (Bs. 499)
ARPU real = Bs. 39.6 + Bs. 110 + Bs. 49.9 = Bs. 199/mes
LTV = 199 / 0.06 = Bs. 3,317
⚖️ LTV/CAC Ratio
LTV/CAC = Bs. 3,317 / Bs. 190 = 17.5x
17.5x
Excelente. Meta industria SaaS: > 3x. > 10x = negocio muy saludable.
Payback Period:
Meses para recuperar CAC = Bs. 190 / Bs. 199 = 0.95 meses ≈ 1 mes
Esto significa que cada cliente es rentable desde el primer mes pagado.

import json from channels.generic.websocket import AsyncWebsocketConsumer from channels.db import database_sync_to_async from ai.parsers.gemini_parser import GeminiOrderParser class StreamConsumer(AsyncWebsocketConsumer): """WebSocket para compradores durante el live.""" async def connect(self): self.stream_id = self.scope['url_route']['kwargs']['stream_id'] self.group_name = f"stream_{self.stream_id}" # Unirse al grupo del live await self.channel_layer.group_add(self.group_name, self.channel_name) await self.accept() # Enviar estado inicial del live stream_data = await self.get_stream_data() await self.send_json({'type': 'stream_state', 'data': stream_data}) async def receive(self, text_data): data = json.loads(text_data) msg_type = data.get('type') if msg_type == 'order_intent': await self.handle_order_intent(data) elif msg_type == 'chat_message': await self.broadcast_chat(data) async def handle_order_intent(self, data): parser = GeminiOrderParser() result = await parser.parse( text=data['text'], stream_context=await self.get_active_products() ) if result['confidence'] >= 0.85: order = await self.create_order(result, data['buyer']) await self.send_json({'type': 'order_confirmed', 'order_number': order.order_number}) # Broadcast al panel del vendedor await self.channel_layer.group_send( f"vendor_{self.stream_id}", {'type': 'new_order', 'order': order.to_dict()} ) else: await self.send_json({'type': 'confirm_order', 'parsed': result})
📄 ai/parsers/gemini_parser.py — Parseo con IA
import google.generativeai as genai import json from django.conf import settings class GeminiOrderParser: """Parsea mensajes de chat en pedidos estructurados usando Gemini.""" def __init__(self): genai.configure(api_key=settings.GEMINI_API_KEY) self.model = genai.GenerativeModel('gemini-1.5-flash') # gratis async def parse(self, text: str, stream_context: dict) -> dict: products_info = json.dumps(stream_context['products'], ensure_ascii=False) prompt = f""" Eres un asistente de ventas boliviano. Analiza este mensaje de un comprador durante un live de ventas y extrae el pedido en formato JSON. Productos disponibles: {products_info} Mensaje del comprador: "{text}" Responde SOLO con JSON válido en este formato: {{ "has_order_intent": true/false, "confidence": 0.0-1.0, "items": [ {{"product_id": "uuid", "variant": "color/talla", "quantity": 1}} ], "notes": "observaciones del parseo" }} Considera variaciones del español boliviano: 'uno pa mi', 'apúntame', 'me llevo', 'quiero', 'dame', 'ponme', 'pido', etc. """ response = self.model.generate_content(prompt) clean = response.text.strip().replace('```json', '').replace('```', '') result = json.loads(clean) return result async def parse_with_fallback(self, text: str, context: dict) -> dict: """Intenta Gemini, usa Groq como fallback.""" try: return await self.parse(text, context) except Exception: from ai.parsers.groq_parser import GroqOrderParser return await GroqOrderParser().parse(text, context)
